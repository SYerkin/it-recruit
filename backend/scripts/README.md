# Скрипты для работы с базой данных

## Импорт мок-данных

Для импорта мок-данных в базу используйте команду:

```bash
npm run import:mock
```

Скрипт:
- Очищает существующие данные
- Создает пользователей (HR и кандидатов)
- Создает компании
- Создает навыки
- Создает профили кандидатов
- Создает связи между кандидатами и навыками
- Создает опыт работы
- Создает образование

**Важно:** 
- Все пароли захешированы через bcrypt
- Пароль для всех пользователей: `admin123`
- Скрипт автоматически очищает базу перед импортом

## Структура данных

После импорта в базе будет:
- 30 пользователей (10 HR + 20 кандидатов)
- 10 компаний
- 20 профилей кандидатов
- 12 навыков
- 9 связей кандидат-навык
- 5 записей опыта работы
- 5 записей образования
