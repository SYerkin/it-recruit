# IT-Recruit — Обзор проекта

## Что это

IT-Recruit — рекрутинговая платформа для IT-специалистов в Казахстане. Соединяет три типа пользователей: кандидатов, HR-менеджеров и менторов. Кандидаты ищут работу и менторов, HR публикуют вакансии и ведут найм, менторы проводят платные сессии.

---

## Технический стек

| Слой | Технологии |
|------|------------|
| **Frontend** | React 18, TypeScript, Vite, Styled Components |
| **Backend** | Node.js, Express.js, Prisma ORM |
| **База данных** | SQLite (dev) / PostgreSQL (prod) |
| **Аутентификация** | JWT + bcryptjs |
| **Файлы** | Multer (загрузка логотипов, документов, резюме) |
| **Карты** | Leaflet + React-Leaflet + OpenStreetMap / Nominatim |
| **Формы** | React Hook Form + Zod |
| **Стейт** | Zustand (авторизация), TanStack React Query (серверные данные) |
| **Анимации** | Framer Motion |
| **Иконки** | Lucide React |
| **API клиент** | Axios с JWT интерцептором |

## Архитектура

```
it-recruit/
├── backend/          # Node.js + Express + Prisma
├── frontend/         # React 18 + TypeScript + Vite
├── uploads/          # Хранилище файлов (логотипы, CV)
└── PLAN.md           # Дорожная карта разработки
```

**Backend** запускается на порту `3000`, **Frontend** на `5173`. В корне есть `concurrently` скрипт (`npm run full`) для одновременного запуска.

### Backend

```
backend/src/
├── app.js              # Express setup (CORS, rate limiting, routes)
├── server.js           # Точка входа
├── controllers/        # 17 контроллеров — вся бизнес-логика
├── routes/             # 16 роут-файлов
├── middlewares/        # JWT auth middleware
└── utils/              # Prisma клиент, стандартные ответы, Zod валидация
```

Все ответы приходят в едином формате через `utils/response.js`. Валидация входных данных через Zod схемы. На auth-эндпоинтах стоит `express-rate-limit`.

### Frontend

Используется архитектурный паттерн **Feature-Sliced Design (FSD)**:

```
frontend/src/
├── app/            # Инициализация приложения, глобальный стор
├── pages/          # 17 страниц (Admin, Jobs, Mentors, Auth и т.д.)
├── widgets/        # Крупные переиспользуемые блоки (Layout, FeedbackButton)
├── features/       # Фичи (feedback modal и т.д.)
├── entities/       # Бизнес-сущности
└── shared/         # API клиенты, UI-компоненты, утилиты
```

## База данных

17 основных моделей в Prisma schema, 15 миграций.

| Модель | Назначение |
|--------|------------|
| `User` | Авторизация, роли: ADMIN / HR / CANDIDATE |
| `CandidateProfile` | Профиль кандидата: навыки, образование, опыт |
| `Company` | Компания HR-а: вакансии, верификация |
| `Job` | Вакансии: зарплата, навыки, дедлайны |
| `Application` | Отклик на вакансию со статусом (APPLIED → HIRED/REJECTED) |
| `ApplicationStatusHistory` | Timeline смены статусов заявки |
| `ApplicationFeedback` | Взаимные оценки после найма (HR ↔ Кандидат) |
| `Conversation` + `ChatMessage` | Чат между HR и кандидатом |
| `Mentor` | Профиль ментора: навыки, цены (0–20 000 ₸) |
| `MentorRequest` | Запрос на менторинг, статус оплаты |
| `MentorReview` | Оценки менторов (1–5) |
| `Payment` | Псевдо-платёжная система (без реального процессинга) |
| `Skill` | Таксономия навыков |
| `RecommendationCategory` + `RecommendationItem` | Образовательные ресурсы |
| `FeedbackMessage` + `MentorApplication` | Формы обратной связи и заявки на менторство |
| `VerificationRequest` | Верификация компаний |

## Роли и доступы

- **CANDIDATE** — поиск работы, отклики, профиль, менторинг
- **HR** — управление вакансиями, воронка найма, профиль компании
- **ADMIN** — полный доступ ко всей платформе

---

## Основные фичи

### Для кандидатов

**Поиск работы**
- Список вакансий с фильтрами: навыки, зарплата (в тенге), тип занятости, компания
- Детальная страница вакансии: описание, требования, счётчик просмотров, похожие вакансии
- Сохранение в избранное

**Отклики и трекинг**
- Подача заявки с сопроводительным письмом
- Timeline статусов: APPLIED → SCREENING → INTERVIEW → OFFER → HIRED/REJECTED
- Чат с HR внутри заявки

**Профиль**
- CV: образование, опыт, навыки, сертификаты
- Публичная страница профиля для HR

**Менторинг**
- Поиск менторов с фильтрами по навыку, цене, рейтингу
- Страница ментора: биография, отзывы, цены
- Запрос сессии с псевдо-оплатой картой

**Обучение**
- Раздел с образовательными ресурсами: курсы, книги, советы по карьере
- Разбито по категориям (React, Backend, Soft Skills и т.д.)

---

### Для HR

**Управление вакансиями**
- CRUD вакансий: название, зарплата, требования, дедлайн
- Статус вакансии (активна / закрыта)

**Воронка найма**
- Kanban-доска для ведения кандидатов по этапам
- Смена статуса заявки прямо в интерфейсе
- Чат с кандидатом

**Профиль компании**
- Логотип, сайт, описание, офисный адрес
- Интерактивная карта (Leaflet + Nominatim геокодинг)
- Значок верификации после проверки

**Аналитика**
- Дашборд: метрики по откликам, наймам, просмотрам вакансий

---

### Для Admin

**Дашборд** — 8 ключевых метрик платформы: пользователи, компании, вакансии, заявки, менторы.

**Управление:**
- Пользователи: просмотр, сброс пароля
- Компании: верификация (одобрить / отклонить)
- Заявки: просмотр всех откликов с фильтрами
- Менторы: CRUD через modal-форму
- Заявки на менторство: review и одобрение
- Обратная связь: просмотр, отметка как прочитанное

## User Flow

### Кандидат

```
Регистрация → Заполнение профиля (навыки, опыт)
    → Поиск вакансий → Просмотр деталей → Подача заявки
    → Ожидание → Смена статуса → Чат с HR
    → Найм / Отказ → Взаимные оценки
    
    Параллельно:
    → Поиск ментора → Запрос сессии → Оплата → Сессия → Отзыв
    → Изучение ресурсов в разделе "Обучение"
```

### HR

```
Регистрация → Создание профиля компании → Верификация
    → Публикация вакансий
    → Входящие заявки → Kanban воронка
    → Смена статусов → Чат с кандидатом
    → Найм / Отказ → Взаимные оценки
```

### Admin

```
Логин → Dashboard (метрики)
    → Верификация компаний
    → Управление менторами и контентом
    → Работа с обратной связью
```

## API — ключевые эндпоинты

**Auth**
```
POST /api/auth/register
POST /api/auth/login
GET  /api/auth/me
PATCH /api/auth/me
POST /api/auth/change-password
```

**Jobs**
```
GET    /api/jobs              — список с фильтрами
GET    /api/jobs/:id          — детали + инкремент просмотров
POST   /api/jobs              — создать (HR)
PUT    /api/jobs/:id          — обновить
DELETE /api/jobs/:id          — удалить
```

**Applications**
```
POST   /api/applications/:id/apply    — подать заявку
GET    /api/applications              — мои заявки
PATCH  /api/applications/:id/status  — изменить статус
POST   /api/applications/:id/feedback — оставить оценку
POST   /api/applications/:id/chat    — отправить сообщение
```

**Mentors**
```
GET  /api/mentors             — список с фильтрами
GET  /api/mentors/:id         — профиль + отзывы
POST /api/mentors/:id/request — запрос сессии
POST /api/mentors/:id/reviews — оставить отзыв
POST /api/payments/process    — оплата сессии
```

**Admin**
```
GET  /api/admin/stats                    — метрики платформы
GET  /api/admin/users                    — все пользователи
POST /api/admin/users/:id/reset-password — сброс пароля
GET  /api/admin/companies                — все компании
GET  /api/admin/feedback                 — обратная связь
```

## Запуск проекта

```bash
# Установка зависимостей
npm run install:all

# Запуск (backend + frontend параллельно)
npm run full

# Только backend (порт 3000)
npm run backend

# Только frontend (порт 5173)
npm run frontend
```

**Backend команды:**
```bash
npm run prisma:migrate   # применить миграции
npm run seed             # заполнить тестовыми данными
npm run prisma:studio    # GUI для базы данных
```
